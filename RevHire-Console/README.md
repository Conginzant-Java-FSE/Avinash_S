# RevHire Console

Welcome to the RevHire Console. This is basically a Java console app for hiring.

## What's this?
It's for:
- Job seekers to find Jobs.
- Employers to post Jobs.

## Features
 User Authentication: Secure login and registration for Job Seekers and Employers.

- Job Management : Employers can post, update, and manage job listings.

- Job Search: Job seekers can browse and search for available positions.

- Application Tracking: Users can apply for jobs and see the status of their applications.

- Profile Management: Maintain and update user profiles and resumes.

- Notifications: You will get the notifications too.

## Technologies used
- Java
- Maven
- MySQL
- JUnit
- Loggers
